Movie Genre Classification

This project was completed at the end of a machine learning course at Georgia Tech. The goal was to train a model that, given the text body of a movie's plot description pulled from Wikipedia, could classify that movie's genre as one of 9 possible options. I manually implemented a classifier based on the Naive Bayes Algorithm using a bag-of-words representation of each movie's plot summary. This model was able to correctly classify the genre of about 60% of movies it was tested on, and it performed as well or better than implementations using packages such as scikit-learn.
